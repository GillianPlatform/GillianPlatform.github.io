type 'result cont_func_f = ?path:Gil_syntax.Branch_case.path -> unit -> 'result cont_func

To support the step-by-step behaviour of the debugger, execution is split into thunks; each invocation executes one GIL command. By supplying a branch path, a particular branch of execution can be selected.

    3. new_branch_cases : Gil_syntax.Branch_case.t list;
    4. cont_func : 'result cont_func_f;
    }
